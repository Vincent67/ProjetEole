méthode getCatalogTerm (SQLServerDatabaseMetaData)

Récupère le terme favori du fournisseur de base de données pour « catalog ».

Syntaxe

public java.lang.String getCatalogTerm()

Valeur de retour

String qui contient le terme du catalogue.

Exceptions

SQLServerException

Notes

Cette méthode getCatalogTerm est spécifiée par la méthode getCatalogTerm dans l'interface java.sql.DatabaseMetaData.

Lors de l'utilisation du Pilote JDBC Microsoft pour SQL Server avec une base de données SQL Server, cette méthode retourne le terme « database ».
