Méthode registerOutParameter (int, int)

Inscrit le paramètre OUT dans la position ordinale spécifiée en fonction du type JDBC.

Syntaxe

public void registerOutParameter(int index,
                                 int sqlType)

Paramètres

index

int qui indique la position ordinale du paramètre.

sqlType

Code de type JDBC, comme défini dans java.sql.Types.

Exceptions

SQLServerException

Notes

Cette méthode registerOutParameter est spécifiée par la méthode registerOutParameter dans l'interface java.sql.CallableStatement.

Depuis la version 3.0 du pilote JDBC SQL Server, lorsque sqlType est de type java.sql.Types.TIME, le comportement de cette méthode est modifié par la propriété de connexion sendTimeAsDatetime (Définition des propriétés de connexion) et SQLServerDataSource.setSendTimeAsDatetime.

Pour plus d'informations, consultez Configuration du mode d'envoi des valeurs java.sql.Time au serveur.
