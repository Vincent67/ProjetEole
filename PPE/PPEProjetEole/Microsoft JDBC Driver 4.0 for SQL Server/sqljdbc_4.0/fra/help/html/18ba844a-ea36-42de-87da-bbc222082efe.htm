Méthode getMinutesOffset (DateTimeOffset)

Retourne le décalage, en minutes, par rapport à l'heure GMT, de cet objet DateTimeOffset.

Syntaxe

public int getMinutesOffset()

Valeur de retour

Décalage en minutes.

Notes

Pour un objet DateTimeOffset représentant le 8 mars 2010, à 11:35:48 -0800, getMinutesOffset retourne la valeur 480.
