Méthode isSigned (SQLServerParameterMetaData)

Récupère les informations déterminant si les valeurs pour le paramètre désigné peuvent être des nombres signés.

Syntaxe

public boolean isSigned(int param)

Paramètres

param

int qui indique l'index de paramètre.

Valeur de retour

true si le paramètre désigné peut contenir des nombres signés. Sinon, false.

Exceptions

SQLServerException

Notes

Cette méthode isSigned est spécifiée par la méthode isSigned dans l'interface java.sql.ParameterMetaData.
