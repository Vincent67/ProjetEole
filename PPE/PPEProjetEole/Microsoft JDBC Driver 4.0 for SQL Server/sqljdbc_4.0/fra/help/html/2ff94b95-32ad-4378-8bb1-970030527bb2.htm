Méthode isFirst (ISQLServerResultSet)

Récupère les informations déterminant si le curseur se trouve sur la première ligne de cet objet SQLServerResultSet.

Syntaxe

public boolean isFirst()

Valeur de retour

true si le curseur se trouve sur la première ligne. false si le curseur se trouve à une toute autre position ou si le jeu de résultats ne contient aucune ligne.

Exceptions

SQLServerException

Notes

Cette méthode isFirst est spécifiée par la méthode isFirst dans l'interface java.sql.ResultSet.

Si cette méthode est utilisée avec les curseurs avant et dynamiques, une exception est levée.
