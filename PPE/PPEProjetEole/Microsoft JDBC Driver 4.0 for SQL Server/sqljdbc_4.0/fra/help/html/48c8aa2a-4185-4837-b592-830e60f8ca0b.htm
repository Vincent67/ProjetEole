Méthode setNClob (int, java.sql.NClob)

Définit le paramètre désigné selon l'objet NClob spécifié.

Syntaxe

public final void setNClob(int parameterIndex,
              java.sql.NClob value)

Paramètres

parameterIndex

int qui indique l'index de paramètre.

value

Objet NClob qui indique la valeur du paramètre.

Exceptions

SQLServerException

Notes

Cette méthode setNClob est spécifiée par la méthode setNClob dans l'interface java.sql.PreparedStatement.
