Méthode getParameterClassName (SQLServerParameterMetaData)

Récupère le nom complet de la classe Java dont les instances doivent être passées à la méthode setObject de la classe SQLServerPreparedStatement.

Syntaxe

public java.lang.String getParameterClassName(int param)

Paramètres

param

int qui indique l'index de paramètre.

Valeur de retour

String qui contient le nom complet de la classe.

Exceptions

SQLServerException

Notes

Cette méthode getParameterClassName est spécifiée par la méthode getParameterClassName dans l'interface java.sql.ParameterMetaData.
