Méthode executeQuery (SQLServerStatement)

Exécute l'instruction SQL donnée et retourne un seul objet SQLServerResultSet.

Syntaxe

public java.sql.ResultSet executeQuery(java.lang.String sql)

Paramètres

sql

String qui contient une instruction SQL.

Valeur de retour

Objet SQLServerResultSet.

Exceptions

SQLServerException

Notes

Cette méthode executeQuery est spécifiée par la méthode executeQuery dans l'interface java.sql.Statement.

Une exception SQLServerException est levée si l'instruction SQL donnée produit tout autre élément qu'un objet SQLServerResultSet unique.

Si vous exécutez les résultats d'une procédure stockée dans plusieurs mises à jour, ou dans une mise à jour qui retourne plus d'un jeu de résultats, vous pouvez utiliser la méthode execute pour exécuter la procédure stockée.
