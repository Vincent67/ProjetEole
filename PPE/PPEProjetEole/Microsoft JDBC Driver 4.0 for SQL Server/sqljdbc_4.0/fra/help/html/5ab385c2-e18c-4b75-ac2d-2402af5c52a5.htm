Méthode getFetchDirection (ISQLServerResultSet)

Récupère la direction d'extraction pour cet objet SQLServerResultSet.

Syntaxe

public int getFetchDirection()

Valeur de retour

int qui indique la direction d'extraction actuelle.

Exceptions

SQLServerException

Notes

Cette méthode getFetchDirection est spécifiée par la méthode getFetchDirection dans l'interface java.sql.ResultSet.

Cette méthode retourne FETCH_FORWARD pour les curseurs avant uniquement, le dernier paramètre défini par un appel à la méthode setFetchDirection pour d'autres types de curseurs, et elle retourne FETCH_UNKNOWN pour ces types de curseur si la méthode setFetchDirection n'a jamais été appelée.
