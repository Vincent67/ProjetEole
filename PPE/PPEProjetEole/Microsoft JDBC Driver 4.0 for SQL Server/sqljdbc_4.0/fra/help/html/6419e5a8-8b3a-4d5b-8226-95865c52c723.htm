Méthode getMoreResults (int)

Passe au résultat suivant de cet objet SQLServerStatement et traite n'importe quel objet de jeu de résultats actuellement ouvert en fonction des instructions spécifiées par le mode donné.

Syntaxe

public final boolean getMoreResults(int mode)

Paramètres

mode

int qui indique comment gérer les objets de jeu de résultats actuellement ouverts. Il doit s'agir de l'une des constantes suivantes :

CLOSE_CURRENT_RESULT

KEEP_CURRENT_RESULT (non pris en charge par le pilote JDBC)

CLOSE_ALL_RESULTS

Valeur de retour

true si le résultat retourné est un jeu de résultats. Sinon, false.

Exceptions

SQLServerException

Notes

Cette méthode getMoreResults est spécifiée par la méthode getMoreResults dans l'interface java.sql.Statement.

Si la méthode getMoreResults est appelée avant la récupération des résultats, elle se comporte comme spécifié par l'argument mode et passe au résultat suivant.

Le pilote JDBC ne prend pas en charge la constante KEEP_CURRENT_RESULT. Si elle est utilisée, une exception est levée.
