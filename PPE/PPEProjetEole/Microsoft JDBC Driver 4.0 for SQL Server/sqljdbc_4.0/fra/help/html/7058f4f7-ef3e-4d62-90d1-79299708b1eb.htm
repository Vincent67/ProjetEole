Constructeur SQLServerClob (SQLServerConnection, java.lang.String)

Initialise une nouvelle instance de la classe SQLServerClob, lors de la spécification d'un objet SQLServerConnection et d'une chaîne de données.

Cette méthode a été déconseillée dans la version 2.0 du pilote JDBC. Utilisez plutôt la méthode createClob de la classe SQLServerConnection.

Syntaxe

public SQLServerClob(SQLServerConnection connection,
                     java.lang.String data)

Paramètres

connection

Objet SQLServerConnection.

data

Données CLOB.
