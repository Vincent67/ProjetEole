Méthode setTime (int, java.sql.Time, java.util.Calendar)

Définit le paramètre désigné selon les valeurs d'heure et de calendrier spécifiées.

Syntaxe

public final void setTime(int n,
                          java.sql.Time x,
                          java.util.Calendar cal)

Paramètres

n

int qui indique le numéro de paramètre.

x

Objet Time.

cal

Objet Calendar.

Exceptions

SQLServerException

Notes

Cette méthode setTime est spécifiée par la méthode setTime dans l'interface java.sql.PreparedStatement.

Depuis la version 3.0 du pilote JDBC SQL Server, le comportement de cette méthode est modifié par la propriété de connexion sendTimeAsDatetime (Définition des propriétés de connexion) et par SQLServerDataSource.setSendTimeAsDatetime.

Pour plus d'informations, consultez Configuration du mode d'envoi des valeurs java.sql.Time au serveur.
