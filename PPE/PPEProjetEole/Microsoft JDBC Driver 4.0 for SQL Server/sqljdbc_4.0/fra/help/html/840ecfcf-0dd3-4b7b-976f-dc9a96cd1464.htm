Méthode recover (SQLServerXAResource)

Obtient une liste des branches de transactions préparées à partir d'un gestionnaire de ressources.

Syntaxe

public javax.transaction.xa.Xid[] recover(int flags)

Paramètres

flags

Valeur int qui peut prendre l'une des valeurs suivantes : XAResource.TMSTARTRSCAN, XAResource.TMENDRSCAN, XAResource.TMNOFLAGS ou XAResource.TMSTARTTRSCAN | XAResource.TMENDRSCAN.

Valeur de retour

Objet Xid.

Exceptions

javax.transaction.xa.XAException

Notes

Cette méthode recover est spécifiée par la méthode recover dans l'interface javax.transaction.xa.XAResource.

Si le paramètre flag n'est ni XAResource.TMSTARTRSCAN ni XAResource.TMSTARTRSCAN | XAResource.TMENDRSCAN, une analyse de récupération doit être en cours.
