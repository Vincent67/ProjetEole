Méthode getDateTimeOffset (int)

Cette méthode a été ajoutée dans le pilote JDBC Microsoft SQL Server version 3.0.

Récupère la valeur du paramètre désigné en tant qu'objet Classe DateTimeOffset dans le langage de programmation Java en fonction de l'index de paramètre.

Syntaxe

public microsoft.sql.DateTimeOffset getDateTimeOffset(int index)

Paramètres

index

Position ordinale du paramètre de base un.

Valeur de retour

Objet Classe DateTimeOffset.

Exceptions

SQLServerException

Notes

Vous pouvez définir une valeur de paramètre Classe DateTimeOffset avec SQLServerCallableStatement.setDateTimeOffset.
