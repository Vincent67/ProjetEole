Méthode setURL (SQLServerDataSource)

Définit l'URL utilisée pour la connexion à la source de données.

Syntaxe

public void setURL(java.lang.String url)

Paramètres

url

String qui contient l'URL.

Notes

Pour des raisons de sécurité, n'incluez pas le mot de passe dans l'URL fournie à la méthode setURL. En effet, les serveurs d'applications Java tiers affichent très souvent la valeur définie pour la propriété URL dans l'interface utilisateur de configuration de la source de données. Utilisez plutôt la méthode setPassword pour définir la valeur du mot de passe. Les serveurs d'applications Java n'affichent pas de mot de passe défini dans leur source de données à l'intérieur de l'interface utilisateur de configuration.

Si la méthode setURL n'est pas appelée avant la méthode getURL, getURL retourne la valeur par défaut « jdbc:sqlserver:// ».
