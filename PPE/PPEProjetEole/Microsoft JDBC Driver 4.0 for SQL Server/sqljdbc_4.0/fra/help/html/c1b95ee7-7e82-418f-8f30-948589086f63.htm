Méthode setNClob (java.lang.String, java.io.Reader, long)

Définit le paramètre désigné selon l'objet Reader spécifié, qui correspond au nombre de caractères spécifié.

Syntaxe

public final void setNClob(java.lang.String parameterName,
              java.io.Reader reader,
              long length)

Paramètres

parameterName

String qui contient le nom du paramètre.

reader

Objet Reader.

length

long qui indique le nombre de caractères dans le flux.

Exceptions

SQLServerException

Notes

Cette méthode doit être utilisée pour les types de données de paramètre NCHAR, NVARCHAR, NTEXT et XML.

Cette méthode setNClob est spécifiée par la méthode setNClob dans l'interface java.sql.CallableStatement.
