Méthode getWarnings (SQLServerResultSet)

Récupère le premier avertissement signalé par des appels sur cet objet SQLServerResultSet.

Actuellement, cette méthode n'est pas prise en charge par le Pilote JDBC Microsoft pour SQL Server. Si elle est appelée, cette méthode retourne toujours une valeur Null.

Syntaxe

public java.sql.SQLWarning getWarnings()

Valeur de retour

Objet SQLWarning.

Exceptions

SQLServerException

Notes

Cette méthode getWarnings est spécifiée par la méthode getWarnings dans l'interface java.sql.ResultSet.
