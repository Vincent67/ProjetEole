Méthode position (java.sql.Blob, long)

Retourne la position d'un modèle spécifié dans le BLOB, en fonction du modèle fourni et de l'index de départ.

Syntaxe

public long position(java.sql.Blob pattern,
                     long start)

Paramètres

pattern

Modèle à rechercher.

start

Index de départ pour la recherche.

Valeur de retour

Valeur long de la position à laquelle le modèle a été trouvé, ou -1 si le modèle n'a pas été trouvé.

Exceptions

SQLServerException

Notes

Cette méthode position est spécifiée par la méthode position dans l'interface java.sql.Blob.
