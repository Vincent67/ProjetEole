Méthode getMaxFieldSize (ISQLServerStatement)

Récupère le nombre maximal d'octets pouvant être retournés pour des valeurs de colonne de caractères ou binaire dans un objet SQLServerResultSet, produit par cet objet SQLServerStatement.

Syntaxe

public final int getMaxFieldSize()

Valeur de retour

int indique le nombre maximal d'octets qu'une colonne peut contenir, ou 0 s'il n'existe aucune limite.

Exceptions

SQLServerException

Notes

Cette méthode getMaxFieldSize est spécifiée par la méthode getMaxFieldSize dans l'interface java.sql.Statement.
