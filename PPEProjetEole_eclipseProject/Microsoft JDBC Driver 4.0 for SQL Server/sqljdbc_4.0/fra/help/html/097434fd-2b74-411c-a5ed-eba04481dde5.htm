Classe SQLServerDataSource

Représente une liste des propriétés spécifiques à la connexion à une base de données SQL Server à l'aide d'un objet SQLServerConnection.

Package : com.microsoft.sqlserver.jdbc

Implémente :ISQLServerDataSource, DataSource, java.io.Serializable, javax.naming.Referenceable

Syntaxe

public class SQLServerDatabaseMetaData

Notes

Cette classe prend en charge la désencapsulation dans la classe SQLServerDataSource, l'interface ISQLServerDataSource et l'interface DataSource. Pour plus d'informations, consultez Wrappers et interfaces.
