Méthode getNString (int)

Récupère la valeur du paramètre NCHAR, NVARCHAR ou LONGNVARCHAR désigné en tant que String dans le langage de programmation Java.

Syntaxe

public final java.lang.String getNString(int parameterIndex)

Paramètres

parameterIndex

int qui indique l'index de paramètre.

Valeur de retour

Objet String.

Exceptions

SQLServerException

Notes

Cette méthode getNString est spécifiée par la méthode getNString dans l'interface java.sql.CallableStatement.
