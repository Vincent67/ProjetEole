Méthode setFloat (SQLServerCallableStatement)

Définit le paramètre désigné selon la valeur float donnée.

Syntaxe

public void setFloat(java.lang.String sCol,
                     float f)

Paramètres

sCol

String qui contient le nom du paramètre.

f

Valeur float .

Exceptions

SQLServerException

Notes

Cette méthode setFloat est spécifiée par la méthode setFloat dans l'interface java.sql.CallableStatement.
