Méthode setPortNumber (SQLServerDataSource)

Définit le numéro de port à utiliser pour communiquer avec SQL Server.

Syntaxe

public void setPortNumber(int portNumber)

Paramètres

portNumber

Valeur int qui contient le numéro de port.

Notes

Le numéro de port correspond au numéro de port TCP/IP qui est utilisé lors de l'ouverture d'une connexion socket à SQL Server. Si la propriété portNumber n'est pas définie, la méthode getPortNumber retourne la valeur par défaut de 1433.

La méthode setPortNumber n'effectue aucune vérification de plage sur la valeur de port transmise. Vous pouvez transmettre un numéro de port qui n'est pas valide, comme 99999, sans déclencher d'erreur.
