Méthode executeUpdate (java.lang.String)

Exécute l'instruction SQL fournie, qui peut être une instruction INSERT, UPDATE, MERGE ou DELETE ; ou une instruction SQL qui ne retourne rien, telle qu'une instruction SQL DDL.

Syntaxe

public final int executeUpdate(java.lang.String sql)

Paramètres

sql

String qui contient l'instruction SQL.

Valeur de retour

int qui indique le nombre de lignes affectées ou 0 en cas d'utilisation d'une instruction DDL.

Exceptions

SQLServerException

Notes

Cette méthode executeUpdate est spécifiée par la méthode executeUpdate dans l'interface java.sql.PreparedStatement.

L'appel de cette méthode entraîne une exception étant donné que l'instruction SQL pour l'objet SQLServerPreparedStatement est spécifiée lors de la création de l'objet.
