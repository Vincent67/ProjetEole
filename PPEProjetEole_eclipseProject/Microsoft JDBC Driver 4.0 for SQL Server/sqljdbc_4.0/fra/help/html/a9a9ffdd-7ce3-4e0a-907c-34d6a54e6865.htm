Méthode getResponseBuffering (ISQLServerStatement)

Récupère le mode de mise en mémoire tampon des réponses pour cet objet SQLServerStatement.

Syntaxe

public final java.lang.String getResponseBuffering()

Valeur de retour

String contenant un full ou un adaptive en minuscules.

Notes

adaptive spécifie une mise en mémoire tampon minimale des données, le cas échéant.

full spécifie la lecture de l'intégralité du jeu de résultats à partir du serveur durant l'exécution.

adaptive est la valeur par défaut du pilote JDBC versions 2.0 et 3.0. full était la valeur par défaut des versions du pilote JDBC antérieures à 2.0.

Pour plus d'informations sur l'utilisation du mode de mise en mémoire tampon des réponses, consultez Utilisation de la mise en mémoire tampon adaptative.
