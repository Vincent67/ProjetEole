Méthode setLoginTimeout (SQLServerDataSource)

Définit le nombre de secondes pendant lesquelles cet objet SQLServerDataSource patiente lors de la tentative de connexion.

Syntaxe

public void setLoginTimeout(int loginTimeout)

Paramètres

loginTimeout

Valeur int qui représente le nombre de secondes à attendre. Zéro indique que le délai d'attente correspond au délai d'attente système par défaut, lequel est de 15 secondes.

Notes

Cette méthode setLoginTimeout est spécifiée par la méthode setLoginTimeout dans l'interface javax.sql.DataSource.
