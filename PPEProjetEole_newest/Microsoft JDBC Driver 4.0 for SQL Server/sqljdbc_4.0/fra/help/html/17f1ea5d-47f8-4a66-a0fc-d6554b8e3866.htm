getBinaryStream (java.lang.String)

Récupère la valeur du paramètre désigné en tant que flux binaire d'octets ininterrompus selon le nom du paramètre.

Syntaxe

public final java.io.InputStream getBinaryStream(java.lang.String paramName)

Paramètres

paramName

String qui indique le nom du paramètre.

Valeur de retour

Objet InputStream.
