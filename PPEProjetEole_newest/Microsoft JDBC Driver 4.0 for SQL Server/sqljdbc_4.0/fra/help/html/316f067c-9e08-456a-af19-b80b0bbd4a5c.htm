Méthode getLoginTimeout (SQLServerDataSource)

Retourne le nombre de secondes pendant lesquelles cet objet SQLServerDataSource doit patienter lors de la tentative de connexion.

Syntaxe

public int getLoginTimeout()

Valeur de retour

Valeur int qui représente le nombre de secondes à attendre.

Notes

Si l'application ne spécifie pas explicitement une valeur de délai d'attente, cette méthode retourne une valeur par défaut de 15 secondes.

Cette méthode getLoginTimeout est spécifiée par la méthode getLoginTimeout dans l'interface javax.sql.DataSource.
