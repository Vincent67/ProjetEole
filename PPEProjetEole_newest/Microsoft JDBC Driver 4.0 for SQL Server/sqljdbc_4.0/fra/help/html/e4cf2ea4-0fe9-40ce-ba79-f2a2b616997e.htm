Méthode compareTo (DateTimeOffset)

Compare cet objet DateTimeOffset à un autre objet DateTimeOffset en fonction de l'heure par rapport à l'heure GMT.

Syntaxe

public int compareTo(DateTimeOffset other)

Paramètres

Valeur DateTimeOffset que vous souhaitez comparer à l'instance actuelle.

Valeur de retour

Le tableau suivant décrit la valeur renvoyée par cette méthode :

Valeur renvoyée

Description

0

Les deux objets DateTimeOffset représente le même point précis dans le temps.

nombre négatif

Cet objet DateTimeOffset représente un point précis dans le temps situé avant other.

nombre positif

Cet objet DateTimeOffset représente un point précis dans le temps situé après other.

Notes

Lorsque deux objets DateTimeOffset ont le même point dans le temps par rapport à l'heure GMT, aucun autre classement des objets basé sur le décalage n'est effectué.
